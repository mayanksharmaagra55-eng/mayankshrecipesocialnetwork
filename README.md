# Recipe Social Network

## Project Overview
Recipe Social Network is a web-based platform where users can explore and interact with recipe content. This website provides a social media style experience for food lovers to discover new recipes and engage with recipe posts.

## Features
- User Login
- Home Page with Featured Recipes
- Explore Recipes Page
- Like Recipe Posts
- User Profile Page
- Responsive Design for Mobile and Desktop

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Forms (Dummy Login Form)

## Project Structure
- `index.html` – Main home page
- `explore.html` – Explore recipes page
- `login.html` – Login page
- `profile.html` – User profile page
- `style.css` – Styling file
- `script.js` – JavaScript functionality
